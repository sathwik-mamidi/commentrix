Overview

Commentrix is a sunset project. This website is a static public archive and does not offer accounts, demos, subscriptions, payments, uploads, or customer support workflows.

Information this site collects

The static Commentrix archive does not intentionally collect personal information. The site does not include a contact form, analytics script, booking widget, login, or payment flow.

Third-party services

The page is intended to be hosted through Cloudflare Pages and embeds a Vimeo demo video. Those providers may receive standard technical request information, such as IP address, browser details, referring page, and playback interactions, according to their own policies.

MVP code

The repository includes local MVP source code for archival reference. Running that code locally may require API keys and a local video file. Do not commit local .env files, generated videos, subtitles, or other private runtime material.

Questions

This archive is provided as-is. No active Commentrix service is available.
